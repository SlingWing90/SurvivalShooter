ParticleEmitter Class

Defined in: C:\MattiusSVN\ParticleEmitterJS\Website\js\ParticleEmitterJS\particleEmitterJs.js:303
Module: createjs

A Particle Emitter extends DisplayObject and must be added to a Container object. An emitter will emit a stream of particles adhereing to the given configuration.
